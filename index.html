Functions

function myFunction(a1, a2) {
                        return a1 * a2;     The function returns the product of a1 and a2
                      };

A JavaScript function is a box of code designed to perform a certain task. A JavaScript function is executed when "something" invokes it (calls it).




Objects

var person = {
                    firstName:"John",
                    lastName:"Doe",
                    age:50,
                    eyeColor:"blue"
                  };

Think of objects as a person. The person has properties like a name, last name, age, and eye colors, and methods are like jump and walk. All people have the same properties, but the property values differ from person to person.





Array

var superheroes = ["Superman", "Iron Man", "Thor"];

An array is a special variable, which can hold more than one value at a time. If you have a list of items (a list of superheroes, for example), storing the superheroes in single variables could look like that. An array can hold many values under a single name superheroes, and you can access the values by referring to an index number starting from zero.
